Old Man in a Cave

Currently enrolled in Video Game Development at triOS or Eastern College

I was 5 years old when I first picked up a controller and somehow beat "The Legend of Zelda" for the first time. This was a tiome before internet, before walktrhoughs, and before youtube tutorials. I spent hours burning bushes and blowing up rock walls hoping to find the secret. Now I find myself falling in love with video games all over again, this time from the design stand point.

The skills and computer languages I'm learning are

Currently I am learning how to code using Visual Studio Code, and this portfolio project has been an introduction into web page development using HTML and Bootstrap.

I'm passionate about the video games I play

Gaming for me has changed a lot over the years. I have "grown" up with the age of gaming side by side, playing each new console that has come out and with them, the next generation of games. With today's current state of "gaming", it has turned into more of a social aspect of my life and am hoping to create new games that friends can join in and play and create lasting memories, not just of the game itself, but of the time had with it.

Freethrow Championship

This was my very first game that I created from scratch using Pico-8. The project's parameters were to include "physics" into our game in some form. It is a simple game in which you use a shot power meter and an angle meter to try and make 2 freethrows to win the game. The project also had to have a win/lose scenario as well. Click here to try!

Bank Robbery

This was the first project I designed and coded using Visual Studio Code. The project was for a Game Jam within our class and had to have any or all of the parameters of "You're the bad guy", "Take one, leave the rest", and "Single player split screen". I decided to do the bad guy and take one route where depending on the loot you decide to take, the guards get faster! Choose wisely!
